Project Explanation:
This project is a Hospital Management System, The system allows users to input doctor and patient information through a Scanner, create an appointment linking them together, and display all relevant details.
Exceptions Implemented:
This project handles potential runtime errors that may occur during user input. The Scanner-based input system can throw an InputMismatchException if the user enters an invalid data type, such as text instead of a number for age or salary. A NullPointerException may occur if a Doctor or Patient object is not properly initialized before being passed to an Appointment.
